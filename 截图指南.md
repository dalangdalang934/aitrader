## 5.1 系统环境搭建

### 5.1.1 服务器环境准备

**步骤1：连接服务器**

```bash
ssh root@your-server-ip
```

**截图点位1**：终端显示成功登录服务器，显示操作系统版本（`lsb_release -a` 或 `cat /etc/os-release`）

---

**步骤2：下载项目代码**

```bash
cd ~
git clone https://github.com/dalangdalang934/aitrader.git
cd aitrader
ls -la
```

**截图点位2**：终端显示git clone成功，项目文件列表

---

### 5.1.2 运行环境安装

**步骤3：执行环境安装脚本**


## 截图注意事项

1. **敏感信息处理**：所有API密钥、密码、私钥必须打码（用黑色块覆盖）
2. **时间显示**：截图中保留系统时间，证明实时性
3. **分辨率**：建议1920x1080或更高，确保文字清晰
4. **浏览器截图**：使用Chrome/Firefox，隐藏书签栏，保持地址栏可见
5. **终端截图**：黑色背景，字体大小适中（推荐14-16px）
6. **图片格式**：界面截图用PNG（清晰），照片用JPG（压缩）
7. **命名规范**：使用英文小写，下划线分隔，如 `dashboard_overview.png`
